Bienvenido al servidor xTx

Operando para tecnicas como: phishing, ingenieria social, transferencia de archivos, demo-pag. etc.

Totrixxel cuenta con una forma de operar muy facil, manejable y simple para todo el publico.

Con las xTx Tools podras cargar cualquier pagina que tengas a el directorio de xTx


Puede descargar cualquier pagina con tan solo el usuario y el nombre del repo en GitHub

Puedes iniciar en cualquier momento tu pagina previamente subida

Puedes montar desde 0 tu pagina sea desde un direcotrio personalizado o desde los prefabs de Totrixxel

Para montar la pagina xTx se requiere la previa instalacion de python

Puedes operar como DNS, para ello debes tener instalado dnsmasq
